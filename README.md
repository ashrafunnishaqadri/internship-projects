## 📁 Projects

### 1. 🎙️ VoxScribe AI
> **Location:** `voxscribe/` &nbsp;|&nbsp; 🌐 **[Live Demo](https://ashrafunnishaqadri.github.io/internship-projects/voxscribe/)**

A premium, high-speed speech-to-text tool that uses real-time Web Speech APIs and on-device AI for both live transcription and audio file processing.

**Features:**
- 🖼️ **Glassmorphic UI**: Premium dark mode theme with neon accents and pulse animations.
- ⚡ **Zero-Lag Transcription**: Live "thinking" text feedback for instant transcription feel.
- 🤖 **On-Device AI (Whisper)**: Local transcription of uploaded audio files using Transformers.js.
- 🌍 **Multi-language Support**: Real-time transcribing for 7+ languages including Hindi and Spanish.
- 📥 **Utility Tools**: Copy to Clipboard, Download as .txt, and Text-to-Speech (Read Aloud).

**Tech Stack:** HTML5 · CSS3 · Vanilla JavaScript · Web Speech API · Transformers.js (Whisper AI)

### 2. 🧠 Text Summarizer
> **Location:** `text-summarizer/` &nbsp;|&nbsp; 🌐 **[Live Demo](https://ashrafunnishaqadri.github.io/internship-projects/text-summarizer/)**

A browser-based intelligent text summarization tool powered by NLP techniques — no backend required.

**Features:**
- 📊 TF-IDF based extractive summarization
- 🔑 Keyword extraction & word frequency analysis
- 📈 Readability statistics (Flesch-Kincaid, etc.)
- ⚡ Fully client-side — works offline
- 🎨 Modern, responsive UI with dark mode

**Tech Stack:** HTML5 · CSS3 · Vanilla JavaScript · NLP Algorithms

## 🚀 How to Run

Each project is standalone. Simply open the `index.html` file in your browser — no installation needed! (Except for AI models, which download automatically).

```bash
# Clone the repository
git clone https://github.com/ashrafunnishaqadri/internship-projects.git

# Navigate into any project
cd internship-projects/voxscribe

# Open in browser
start index.html
```
